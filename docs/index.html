Claw-ED

Open source CLI agent for teachers. Python. MIT licensed.

Install Features Voice Learning Providers Commands Contributing GitHub

Claw-ED is a command-line tool that helps teachers generate lesson plans, assessments, slides, interactive games, and more from their own curriculum files. It learns your teaching style from your existing materials and produces content that sounds like you wrote it.

It works with multiple AI providers (Anthropic, OpenAI, Google, Ollama) so you can pick what fits your budget. Your files stay on your machine. The only external call is to the AI provider you choose.

Requires Python 3.10+. Works on macOS, Windows, and Linux.

Install

# install from PyPI
$ pip install clawed

# run the setup wizard (picks your provider, creates workspace)
$ clawed setup

# ingest your curriculum files
$ clawed ingest ./my-lessons/

# generate a lesson
$ clawed lesson "The Civil War"

The setup wizard is interactive. It walks you through provider selection, API key configuration, and workspace creation. If you have curriculum files ready, it'll ingest those too.

Features

  • Voice learning -- ingests your files, extracts your teaching style, generates content that sounds like you
  • Multi-provider -- Anthropic, OpenAI, Google Gemini, Ollama (local or cloud), OpenRouter
  • 22+ file format ingestion -- PDF, DOCX, PPTX, DOC, PPT, XLS, HTML, Markdown, images, and more
  • Lesson packages -- generates plan + slides + handout + game in one command
  • Interactive HTML games -- AI-generated HTML5 games tailored to each lesson, not templates
  • 50-state standards alignment -- CCSS, NGSS, C3, and state-specific standards databases built in
  • Quality gates -- every output scored 0-10 before delivery; catches truncated text, answer key leaks, broken HTML
  • Telegram bot -- run clawed bot and generate lessons from your phone
  • Agentic onboarding -- the setup wizard uses AI to guide you through configuration step by step
  • Proactive suggestions -- Ed notices patterns in your usage and suggests relevant content
  • Google Drive integration -- pull files directly from Drive for ingestion
  • Browser-based session viewer -- review and edit generated content in a local web UI
  • Data stays local -- no accounts, no telemetry, no data collection

How voice learning works

When you run clawed ingest, Ed reads through your existing lesson plans, worksheets, assessments, and slides. It extracts:

  • Your vocabulary choices and phrasing patterns
  • How you structure lessons (warm-ups, activities, exits)
  • Your question framing style (Socratic, inquiry-based, direct, etc.)
  • Scaffolding moves you use repeatedly
  • Assessment preferences (multiple choice, DBQs, open response)

This creates a voice persona stored locally on your machine. Every time Ed generates content, it applies that persona so the output matches how you actually teach. A physics teacher who relies on worked examples and checkpoint questions will get different output than a history teacher who leads with primary sources and Socratic seminars.

You can have multiple personas (e.g., one per course). The ingest command can be pointed at different folders to build them. 

# ingest AP History files into one persona
clawed ingest ./ap-history/ --persona "AP-Rivera"

# ingest regular History files into another
clawed ingest ./history-10/ --persona "Gen-Rivera"

# generate with a specific persona
clawed lesson "The New Deal" --persona "AP-Rivera"

AI providers

Claw-ED is BYOK (bring your own key). You pick a provider during setup. Switch anytime. Here's what's available:

Provider Models Notes
Ollama Pro Gemma 4, Llama 4, Qwen 3 ~$20/mo. Good balance of quality and cost. Recommended
Anthropic Claude Opus 4.6, Sonnet 4.6 Best voice matching and pedagogy. Higher cost per lesson.
OpenAI GPT-5.4, o3 Strong, reliable output. Also higher cost.
Google Gemini 2.5 Flash, Pro Free tier available. Good way to try things out.
Ollama Local Any supported model Free. Runs fully offline. Quality depends on your hardware.
OpenRouter Any model One API key, access to everything. Pay per token.

For most teachers, Ollama Pro at ~$20/month hits the sweet spot. If you want the highest quality voice matching and don't mind paying more per generation, Anthropic or OpenAI are the way to go. Google's free tier is fine for trying things out.

Commands

clawed setup                  # interactive setup wizard
clawed ingest <dir>           # ingest curriculum files
clawed lesson <topic>         # generate a lesson package
clawed game <topic>           # generate an interactive HTML game
clawed slides <topic>         # generate a slide deck
clawed assessment <topic>     # generate an assessment
clawed unit <topic>           # generate a full unit plan
clawed chat                   # open-ended conversation with Ed
clawed bot                    # start Telegram bot
clawed export <format>        # export to PDF, DOCX, PPTX
clawed standards <state>      # browse standards by state
clawed config                 # view/edit configuration
clawed provider <name>        # switch AI provider

Most commands accept --grade, --standards, --persona, and --format flags. Run clawed <command> --help for details.
