How it works

Three steps. Five minutes. Then EDUagent works for you.

1

Install

One command and you're ready. Works with any Python 3.9+ environment.

pip install eduagent
2

Ingest

Point it at your lesson folders. EDUagent learns your voice, style, and standards.

eduagent ingest ./lessons
3

Generate

Generate full units, lessons, worksheets, PDF handouts, slides, or assessments on demand.

eduagent lesson "WW2" --export pdf

Try it now — completely free

100% open source. No account, no subscription, no credit card. Install in minutes on any Mac, PC, or Linux computer.

1
Install Python (if you don't have it) — python.org/downloads. Click the big button and run the installer.
2
Open Terminal — On Mac: press Cmd + Space, type Terminal, hit Enter. On Windows: press the Windows key, type cmd, hit Enter.
3
Install EDUagent — paste this and press Enter: pip install eduagent
4
Start it up: eduagent chat EDUagent will walk you through the rest.
